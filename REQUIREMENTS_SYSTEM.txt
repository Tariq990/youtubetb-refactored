===============================================
  YouTubeTB - متطلبات النظام الكاملة
===============================================

📌 Python:
- النسخة المثبتة حالياً: Python 3.13.7
- pip (يأتي مع Python)

📌 FFmpeg:
- لمعالجة الفيديو والصوت
- التثبيت: تلقائي عبر SETUP_ALL.bat
- أو يدوي من: https://ffmpeg.org

📌 Playwright:
- متصفح Chromium
- للـ Web Scraping
- التثبيت: تلقائي عبر السكريبت

📌 Git:
- لتنزيل المشروع من GitHub
- تنزيل: https://git-scm.com/download/win

📌 اتصال إنترنت:
- لتنزيل المكتبات والأدوات
- سرعة موصى بها: 5 Mbps أو أعلى

📌 مساحة القرص:
- ~2-3 GB للتثبيت الكامل
  - Python: ~100 MB
  - FFmpeg: ~100 MB
  - المكتبات: ~500 MB
  - Playwright: ~300 MB
  - الباقي للملفات المؤقتة

📌 RAM:
- الحد الأدنى: 4 GB
- الموصى به: 8 GB أو أكثر

📌 المعالج:
- الحد الأدنى: Intel i3 أو ما يعادله
- الموصى به: Intel i5 أو أفضل (للمعالجة الأسرع)

===============================================
  API Keys & Secrets المطلوبة
===============================================

📌 Gemini API Key (إجباري):
- للذكاء الاصطناعي (ترجمة، معالجة نصوص)
- الحصول عليه: https://makersuite.google.com/app/apikey
- ملف: secrets/api_key.txt

📌 YouTube OAuth (إجباري للرفع):
- client_secret.json
- token.json (يتم إنشاؤه تلقائياً)
- من: https://console.cloud.google.com/

📌 YouTube Cookies (اختياري):
- للفيديوهات المحظورة حسب العمر
- ملف: secrets/cookies.txt
- استخراج: via browser extension

📌 Pexels API Key (اختياري):
- لفيديوهات الخلفية في Shorts
- من: https://www.pexels.com/api/
- ملف: secrets/pexels_key.txt أو في .env

===============================================
  أدوات خارجية
===============================================

📌 ffmpeg:
- تثبيت تلقائي في: C:\ffmpeg
- أو يدوي من: https://ffmpeg.org

📌 ffprobe:
- يأتي مع FFmpeg
- للحصول على معلومات الصوت/الفيديو

📌 yt-dlp:
- يثبت عبر pip تلقائياً
- لتنزيل فيديوهات YouTube

📌 Playwright Chromium:
- يثبت عبر: playwright install chromium
- للـ Web Scraping (OpenAI.fm TTS)

===============================================
  الملفات الإجبارية في المشروع
===============================================

📌 في الجذر:

- api_key.txt
- client_secret.json
- cookies.txt (اختياري)
- token.json (يُنشأ تلقائياً)

📌 في src/:
- كل أكواد المشروع

📌 في scripts/:
- سكريبتات مساعدة
- encrypt_secrets.py
- decrypt_secrets.py
- test_*.py

===============================================
  أذونات Windows المطلوبة
===============================================

- لتثبيت Python

- للقراءة/الكتابة في مجلد المشروع
