# ============================================
# NOTES
# ============================================

# 📝 Matching Logic:
# - System uses case-insensitive title + author matching
# - "Atomic Habits | James Clear" matches "atomic habits | james clear"
# - Books without author: matches ONLY by title

# ⚡ Performance:
# - Process 10-20 books per batch (recommended)
# - Each book takes 10-60 minutes (depends on video length)
# - Total batch time: multiply book count × average time

# 🔄 Resume System:
# - If interrupted, re-run with same books.txt
# - System will skip completed and resume incomplete
# - No duplicate work!

# 🎯 Best Practices:
# - Keep books.txt organized by category
# - Use comments (#) for readability
# - Include author names for accuracy
# - Process similar books together (same playlist)
